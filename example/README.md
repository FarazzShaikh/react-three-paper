  = `App.js`:
    - Connecting ThreeJS and React using `react-three-paper`

## To run

```bash
$ git clone https://github.com/FarazzShaikh/react-three-paper.git
$ cd examples
$ yarn install
$ yarn start # http://localhost:5000
```
